## 📐 DAX Measures

All **9 custom DAX measures** created and used in this dashboard:

---

### 1. 🔢 Total Incidents
> **Used on:** Global Overview · Attack Analysis · Impact & Response

```dax
Total Incidents = COUNTROWS('Global_Cybersecurity_Threats_2015-2024')
```

---

### 2. 👥 Total Users Affected
> **Used on:** Global Overview · Impact & Response

```dax
Total Users Affected = 
    SUM('Global_Cybersecurity_Threats_2015-2024'[Number of Affected Users])
```

---

### 3. 💰 Avg Financial Loss
> **Used on:** Global Overview · Impact & Response

```dax
Avg Financial Loss = 
    AVERAGE('Global_Cybersecurity_Threats_2015-2024'[Financial Loss (in Million $)])
```

---

### 4. 💸 Avg Financial Loss Per Attack
> **Used on:** Attack Analysis — KPI Card · Bar Chart · Scatter Plot

```dax
Avg Financial Loss Per Attack = 
    AVERAGE('Global_Cybersecurity_Threats_2015-2024'[Financial Loss (in Million $)])
```

---

### 5. ⏱️ Avg Resolution Time
> **Used on:** Global Overview · Attack Analysis · Impact & Response

```dax
Avg Resolution Time = 
    AVERAGE('Global_Cybersecurity_Threats_2015-2024'[Incident Resolution Time (in Hours)])
```

---

### 6. 📊 Attack Frequency %
> **Used on:** Attack Analysis — KPI Card

```dax
Attack Frequency % = 
DIVIDE(
    COUNT('Global_Cybersecurity_Threats_2015-2024'[Attack Type]),
    CALCULATE(
        COUNT('Global_Cybersecurity_Threats_2015-2024'[Attack Type]),
        ALL('Global_Cybersecurity_Threats_2015-2024'[Attack Type])
    ),
    0
)
```

---

### 7. 🎯 Most Frequent Attack Type
> **Used on:** Attack Analysis — KPI Card

```dax
Most Frequent Attack Type = 
FIRSTNONBLANK(
    TOPN(
        1,
        VALUES('Global_Cybersecurity_Threats_2015-2024'[Attack Type]),
        CALCULATE(COUNT('Global_Cybersecurity_Threats_2015-2024'[Attack Type])),
        1
    ),
    1
)
```

---

### 8. 🏭 Most Targeted Industry
> **Used on:** Impact & Response — KPI Card

```dax
Most Targeted Industry = 
FIRSTNONBLANK(
    TOPN(
        1,
        VALUES('Global_Cybersecurity_Threats_2015-2024'[Target Industry]),
        CALCULATE(COUNT('Global_Cybersecurity_Threats_2015-2024'[Target Industry])),
        0
    ),
    1
)
```

---

### 9. 🔧 Hardest To Resolve
> **Used on:** Attack Analysis — supporting insight

```dax
Hardest To Resolve = 
FIRSTNONBLANK(
    TOPN(
        1,
        VALUES('Global_Cybersecurity_Threats_2015-2024'[Attack Type]),
        CALCULATE(
            AVERAGE('Global_Cybersecurity_Threats_2015-2024'[Incident Resolution Time (in Hours)])
        ),
        1
    ),
    1
)
```

## 🚀 Getting Started

### Prerequisites
- [Power BI Desktop](https://powerbi.microsoft.com/desktop/) installed on your machine

### Steps

```bash
# 1. Clone the repository
git clone https://github.com/Sahilmore469/CyberThreat-Intelligence-Dashboard.git

# 2. Navigate to the project folder
cd CyberThreat-Intelligence-Dashboard
```

```
# 3. Open in Power BI Desktop
   → File → Open → CyberThreat_Dashboard.pbix

# 4. Refresh Data Source (if prompted)
   → Home → Refresh

# 5. Explore!
   → Use slicers on the left panel to filter and interact with the data
```
